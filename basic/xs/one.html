# 第零章 长生引

  多年之后。星辰海畔。吟游诗人跌跌撞撞，慨然高歌。

  歌曰：

雁过惊天逞虚妄，箜篌声断洒清秋。

宜将悲情唱离恨，莫使韶华付青鸾。

五谷蜀黍觉味浅，倏忽一日起变奏。

九天神雷降华颠，冥漱玉兮风吹衣。

魂追轩辕战涿鹿，笑看霸王殒江东。

意属长城戍倥偬，愿以碧血染红缨。

剑名长生诛神仙，可与北辰共争辉。

斗笠麻衣煞北风，一人一剑斩崔嵬。

长生剑客谁得见？随风到处无人还。

北邙山中葬北邙，长生剑下断长生。

# 第壹章 叹大唐长安 宿劫天降

  他推动那扇锈迹斑斑的铁门，锈蚀的门轴发出沙哑而沉重的叹息。门应声而开。当他不自觉地循着那一缕光亮向前走去，突然被一种玄妙的力量绊倒，意识顿时陷入无尽的黑暗。

  再一睁眼时，少年已经出现在了大唐长安城的市集上。

  道路宽阔却不洁净，楼阁高耸而亦老旧。朱漆剥蚀了大半，瓦片残缺参差不齐。少年独自走在街市上，感受着沾满鞋底的污泥，以及袭面的夹杂着雨腥味的清风。

​  微风飕飗。

​  他忽尔发现了一件极其诡异之事：哪怕是在人来人往的集市上，这里仍然死气沉沉，空气凝胶般的粘稠。“烧——饼——烧——饼——”街边的叫卖声像是台万年不变的留声机，重复播放着一个又一个平凡的音节，无抑扬，无顿挫，石头扔进去也不会激起半点波澜。人们面黄肌瘦，脸上写满了灰暗，目中充斥着绝望。

​  这时少年眼角余光一瞟，无意中注意到了街角那个趿着屐的老头。不知为何，一种悚然的心悸之感掠过他的心头。他又感觉自己仿佛在哪里见过此人。老头拄着一根两人高的竹竿，上挂白幡，隐约可见字迹，却看不真切其上所书为何。目光向下，只见这白幡下站着一个小女孩，叼一根棒棒糖，顶两根冲天发辫，间或吸溜一下鼻涕。老头若有若无地向少年这边看了一眼，忽然喟然长叹：“唉！世风日下……人心不古啊！”

​  倏忽一刹那，滚滚黑云覆满了苍穹，犹如千军万马奔袭而来。人们的脚步开始变得急促，啪嗒，啪嗒，如梵魂圣音叩击在少年心魂深处，少年的意识因此而激荡，灵魂因此而震颤。他的脑海深处像是出现了一个无底的漩涡，要将他的意识拖入其中，然后狂暴地撕裂。只觉眼前一黑，少年猛然惊醒，不知不觉间冷汗已经浸透了他的长衫。

​  诡异之感丝毫没有消退。他抬头一看，街角空空荡荡。他不自觉地握紧了剑柄……

# 第贰章 待铁门之后 世断纷争

​  门应声而开。

​  眼前是一片血色的修罗地狱，断肢堆积成山，腥血逆流成河。在真正压倒性的残忍面前，任何描述的语言都显得苍白而无力。血色的天，血色的月，倾泻下血色的光辉。无尽的尸山中仿佛有修罗即将出世，遥远的天边有漫天神魔低吼。

​  可无论修罗神魔，他相信，在席卷这片土地的金戈铁马面前，必将被碾成微尘，无数凡人担着他们毕生的荣耀，拿出最后的勇武，必将打出一场弑神的战争。

​  过往的记忆如潮水般涌入脑海，他头痛几欲炸裂，死死地咬紧牙关不让惨嚎迸发。恨，突然弥漫在他的心中，是那种不可抗拒的恨，充斥了他身上的每一个角落。

​  突然，血月降下了光，直中他的身躯，仿佛要将这千万人的力量集中于他一身。

​  天降的恩赐让他措手不及。那一刹那，他能感受到身体中流淌着的力量，那是一种奔涌在血脉中的蛮荒神力，也是一种超脱层面的力量，是天道之力。手可摘星辰，脚可踏日月。

​  有道是：

​  善则天下永安，戾则魔神戮世。

​  心怀苍生，恨满盈野。

​  他背负千万人的恨，要去拯救亿万人。

# 第叁章 愿天下人间 永无饥馁

​  天道给予他的，既是创世之力，亦是灭世之力。殊不知天道与他，有着共同的归宿。

​  田间地头，秋高气爽，天边云游一道人。

​  三十年前，这里还是一片焦土，为战争所洗礼，寸寸土地都曾被鲜血浸透。你或许不敢相信，但这黑发道人真真切切地弹指可灭天下。

​  道人走到茅屋前，伸手轻轻地在梁柱上摩挲，感受着原木上的细致纹路，似要触及内心深处的柔软。屋内适时走出了一小儿，面容稚气未脱，头上盘着一个发髻，抬头看向道人，眼神清澈得如无暇的玉石，映着无垢的光芒。他一张口发出轻灵的声音，胜过任何天籁：“前辈何许人也？”

​  道人呵呵一笑，却笑遍了世间沧桑。面容虽是中年，却仿佛经历了人生百世。他又自嘲一笑，说：“我是谁？呵呵……我还是我吗？”

​  唉，不重要了。

​  道人仔细端详着小儿的脸，突然心中一震。

​  呈现在他眼前的，是一张清瘦的面孔，纵然俊美也难掩其枯弱。

​  他又转头望向无尽的远方，但并没有看见铁骑雄师，也没有看见漫天黄沙。与天接壤之处，是同样枯瘦的土地，零零星星地点缀着几根麦穗。

​  罢了。俱往矣。

​  中年道人心念一动，白幡迎风招展。其上四个大字赫然：“神机”“天算”。他一挥长袖，整个人突兀消失不见，如水滴入水中。时隔百年，山头再次葱郁，一绺白发随风飘扬。

# 第肆章 逢烟火盛世 道洽大同

​  游荡的孤高灵魂不需要众神的垂悯，因为他没有影子。

​  雨滴银蛇般狂舞，织成密不透光的雨幕，其中仿佛有雷龙在游弋，时不时炸响凛然天威。

​  白墙，赤瓦，青苔，黑影。

​  一道矫健的身影灵活地在屋檐上跳跃、腾挪。脚尖点在砖瓦上的噼啪声被雨声完全盖过，他无声无息，像是雨中的幽灵。他突然跃起，如一柄利剑，刺破苍穹。

​  他重重地砸下，哪怕是皇城的大道亦扛不住冲击力而寸寸粉碎，扬起一阵碎石的风暴。在雨中，他直起了身，然后转身。

​  他眼前出现了一老一少两道身影，老头手上仍然擎着白幡，女孩依然叼着那根棒棒糖。

​  “你们……所为何人？”少年那与年纪不符的富有磁性的嗓音从黑袍下传出，不带半分犹豫。

​  老头亦是转身，把背影留给他，似是不屑，不紧不慢地向前走去。直到老头的身形逐渐消失在巷尾，雨幕中仿佛有歌声传出：我有锋刃……释旧恨济世—— 击楫中流……挽泱泱大同……

​  不知为何，歌声明明那般轻微，却每一声，都重重地砸在少年的心弦之上。晶莹的泪水无声滑落，划过光滑的面庞。这一次他没有再去控制，选择了放浪。

​  “为什么……泱泱大同……这难道真的是宿命……”

# 终章 长生剑下 永断长生

​  他挣扎了一生，所见不过自己的影子；踽踽独行，终不曾跃出天道的囹圄。

​  黑袍走进道观，这显然是一间年久失修的道观，墙边野草疯长，瓦砾碎石遍地都是。但是他丝毫没有注意到这些，似是下了某些坚定的决心，他握紧了剑柄。

​  转至前堂，黑袍的到来没有任何声响。黑袍下的眼睛盯着静静跪在空地上的老头，老头手中攥着三支燃至一半的香，似是在祭拜，面前却空无一物。老头脚边仍摆着他的白幡，已经污浊得不成样子，但隐约还能看见其上的三个字。

​  这一次他没有再犹豫。

​  一声戗然悲鸣，光寒十四州。

​  三根香悄然落地。

​  可是少年再也没有机会注意到，那扇铁门再次在他身后开启，深邃的黑暗吞没了他的长生。
